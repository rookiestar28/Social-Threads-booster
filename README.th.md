# AK-Threads-Booster

> **English Summary:** AK-Threads-booster is a Claude Code and Codex skill and AI writing assistant built specifically for Threads creators. This open-source Threads skill analyzes your historical post data, leverages social media psychology research and the Threads algorithm to provide personalized writing analysis, Brand Voice profiling, and draft assistance. Works as a skill / plugin for Claude Code, Cursor, Codex, Windsurf, GitHub Copilot, and Google Antigravity.

AI Skill สำหรับครีเอเตอร์ Threads โดยเฉพาะ ขับเคลื่อนด้วยข้อมูล รองรับ Claude Code, Cursor, Codex, Windsurf, GitHub Copilot, Google Antigravity วิเคราะห์ข้อมูลโพสต์ย้อนหลังของคุณ ใช้งานวิจัยด้านจิตวิทยาโซเชียลมีเดียและ Threads อัลกอริทึม เพื่อให้การวิเคราะห์การเขียนแบบเฉพาะบุคคล, การสร้าง Brand Voice และการช่วยร่างโพสต์

คอมมูนิตี้ Threads ในประเทศไทยโตเร็วมาก คนไทยชอบคอนเทนต์ที่เข้าถึงง่าย สนุก มีส่วนร่วมได้ แต่คำถามที่ครีเอเตอร์หลายคนยังหาคำตอบไม่เจอคือ โพสต์แบบไหนถึงจะเข้าถึงคนได้มากขึ้น, Threads อัลกอริทึม ทำงานยังไง, เพิ่มผู้ติดตาม Threads ต้องทำอะไร, และเวลาไหนโพสต์แล้วได้ผลดีที่สุด

AK-Threads-Booster ตอบคำถามพวกนี้จากข้อมูลจริงของคุณ ไม่ใช่เทมเพลตสำเร็จรูป ไม่ใช่สูตรสำเร็จจากอินเทอร์เน็ต แต่เป็นระบบที่วิเคราะห์ข้อมูลจากบัญชีของคุณ บอกว่าอะไรใช้ได้ผล และอธิบายเหตุผลจากมุมจิตวิทยาและอัลกอริทึม ถ้าคุณกำลังมองหา Threads เทคนิค ที่ใช้ได้จริง เครื่องมือนี้ออกแบบมาเพื่อสิ่งนี้

## AK-Threads-Booster คืออะไร

AK-Threads-Booster เป็น open-source Threads Skill ไม่ใช่เทมเพลตเขียนโพสต์ ไม่ใช่ชุดกฎ ไม่ใช่ AI เขียนแทนคุณ

เป็นระบบ Skill ที่ติดตั้งแล้วใช้ได้เลย ทำ 3 อย่างหลัก:

1. **วิเคราะห์ข้อมูลย้อนหลังของคุณ** หาว่าคอนเทนต์แบบไหนทำผลงานได้ดีที่สุดในบัญชีของคุณ
2. **ใช้จิตวิทยาและความรู้เรื่อง Threads อัลกอริทึม เป็นเลนส์วิเคราะห์** อธิบายว่าทำไมถึงได้ผล
3. **แสดงผลการวิเคราะห์อย่างโปร่งใส** ให้คุณตัดสินใจเองว่าจะทำอะไรต่อ

ผลลัพธ์ที่ได้จะไม่เหมือนกันสำหรับแต่ละคน เพราะกลุ่มเป้าหมาย สไตล์การเขียน และข้อมูลของแต่ละคนต่างกัน นี่คือความแตกต่างระหว่างเทมเพลตทั่วไปกับกลยุทธ์ Threads ที่ขับเคลื่อนด้วยข้อมูล

## หลักการสำคัญ

**เป็นที่ปรึกษา ไม่ใช่ครู** AK-Threads-Booster ไม่ได้บอกว่า "คุณควรเขียนแบบนี้" แต่จะบอกว่า "ตอนที่คุณเขียนแบบนี้ก่อนหน้านี้ ข้อมูลออกมาแบบนี้ ลองพิจารณาดู" ไม่ให้คะแนน ไม่แก้ไข ไม่เขียนแทน

**ขับเคลื่อนด้วยข้อมูล ไม่ใช่กฎ** ทุกข้อเสนอแนะมาจากข้อมูลย้อนหลังของคุณเอง ไม่ใช่ "10 เคล็ดลับ SNS" ทั่วไป ถ้าข้อมูลไม่พอ ระบบจะบอกตรงๆ ไม่แกล้งทำเป็นมั่นใจ

**เส้นแดงเป็นกฎตายตัวเดียว** เฉพาะพฤติกรรมที่อัลกอริทึมของ Meta ลงโทษชัดเจน (engagement bait, clickbait, โพสต์ซ้ำที่คล้ายกันมาก ฯลฯ) เท่านั้นที่จะเตือนโดยตรง นอกนั้นทุกการวิเคราะห์เป็นแค่ข้อมูลประกอบการตัดสินใจ คุณมีสิทธิ์ตัดสินใจสุดท้ายเสมอ

### ความแตกต่างของฟีเจอร์

- **Claude Code**: ฟีเจอร์ครบ ได้แก่ ตั้งค่าเริ่มต้น(setup), สร้าง Brand Voice(voice), วิเคราะห์โพสต์(analyze), แนะนำหัวข้อ(topics), ช่วยร่างโพสต์(draft), ทำนายผลลัพธ์(predict), รีวิวหลังโพสต์(review) รวม 7 Skill อิสระ
- **เครื่องมืออื่น**: วิเคราะห์การเขียนหลัก 4 มิติ (เทียบสไตล์, วิเคราะห์จิตวิทยา, ตรวจอัลกอริทึม, ตรวจจับ AI) ใช้ knowledge base ชุดเดียวกัน (`knowledge/` directory)

ทุกเวอร์ชันมี:
- แนวทางน้ำเสียงที่ปรึกษา (ไม่ให้คะแนน, ไม่แก้ไข, ไม่เขียนแทน)
- กฎเส้นแดงอัลกอริทึม (ตรงกันเมื่อไหร่เตือนทันที)
- อ้างอิง knowledge base (จิตวิทยา, อัลกอริทึม, ตรวจจับ AI)

## วิธีติดตั้ง

### วิธีที่ 1: ติดตั้งผ่าน GitHub

```bash
# ใน directory โปรเจกต์ Claude Code ของคุณ
claude install-plugin https://github.com/akseolabs-seo/AK-Threads-booster
```

### วิธีที่ 2: ติดตั้งเอง

1. Clone repo มาที่เครื่อง:
   ```bash
   git clone https://github.com/akseolabs-seo/AK-Threads-booster.git
   ```

2. คัดลอก directory `AK-Threads-booster` ไปที่ `.claude/plugins/` ของโปรเจกต์ Claude Code:
   ```bash
   cp -r AK-Threads-booster /path/to/your/project/.claude/plugins/
   ```

3. รีสตาร์ท Claude Code ระบบจะตรวจพบ Skill โดยอัตโนมัติ

### เครื่องมืออื่น

ถ้าใช้ Cursor, Windsurf, Codex หรือ GitHub Copilot แค่ clone repo มาที่ directory โปรเจกต์ของคุณ แต่ละเครื่องมือจะอ่านไฟล์ตั้งค่าที่เกี่ยวข้องโดยอัตโนมัติ

## การตั้งค่าเริ่มต้น

ก่อนใช้งานครั้งแรก ต้องรัน setup เพื่อนำเข้าข้อมูลย้อนหลัง:

```
/setup
```

ระบบจะพาคุณผ่านขั้นตอนต่อไปนี้:

1. **เลือกวิธีนำเข้าข้อมูล**
   - Meta Threads API (ดึงอัตโนมัติ)
   - Meta account export (ดาวน์โหลดเอง)
   - ใส่ไฟล์ข้อมูลที่มีอยู่แล้วโดยตรง

2. **วิเคราะห์โพสต์ย้อนหลังอัตโนมัติ** สร้าง 3 ไฟล์:
   - `threads_daily_tracker.json` -- ฐานข้อมูลโพสต์ย้อนหลัง
   - `style_guide.md` -- คู่มือสไตล์เฉพาะบุคคล (แนวโน้ม Hook, ช่วงจำนวนคำ, รูปแบบการปิดโพสต์ ฯลฯ)
   - `concept_library.md` -- คลังแนวคิด (ติดตามแนวคิดที่คุณเคยอธิบายให้กลุ่มเป้าหมายแล้ว)

3. **รายงานการวิเคราะห์** แสดงลักษณะสไตล์ของบัญชีและภาพรวมข้อมูล

ตั้งค่าเริ่มต้นแค่ครั้งเดียว หลังจากนั้นข้อมูลจะอัปเดตต่อเนื่องผ่านโมดูล `/review`

## 7 Skill

### 1. /setup -- ตั้งค่าเริ่มต้น

รันครั้งแรกเมื่อเริ่มใช้งาน นำเข้าโพสต์ย้อนหลัง สร้างคู่มือสไตล์ สร้างคลังแนวคิด

```
/setup
```

### 2. /voice -- สร้าง Brand Voice

วิเคราะห์เชิงลึกโพสต์ย้อนหลังและคอมเมนต์ตอบกลับทั้งหมด สร้างโปรไฟล์ Brand Voice อย่างละเอียด ลึกกว่าคู่มือสไตล์จาก `/setup` ครอบคลุมความชอบโครงสร้างประโยค, การเปลี่ยนโทน, การแสดงอารมณ์, สไตล์อารมณ์ขัน, คำต้องห้าม และลักษณะเฉพาะอื่นๆ

```
/voice
```

ยิ่ง Brand Voice สมบูรณ์มากเท่าไหร่ ผลลัพธ์จาก `/draft` ก็ยิ่งเหมือนสไตล์เขียนจริงของคุณมากขึ้น แนะนำให้รันหลังจาก `/setup`

มิติการวิเคราะห์: ความชอบโครงสร้างประโยค, รูปแบบการเปลี่ยนโทน, วิธีแสดงอารมณ์, วิธีนำเสนอความรู้, ความแตกต่างของโทนต่อแฟนคลับ vs ต่อคนวิจารณ์, อุปมาอุปไมยที่ใช้บ่อย, สไตล์อารมณ์ขัน, การเรียกตัวเองและผู้อ่าน, คำต้องห้าม, จังหวะย่อหน้า, ลักษณะโทนในการตอบคอมเมนต์

ผลลัพธ์: `brand_voice.md` โมดูล `/draft` จะอ้างอิงโดยอัตโนมัติ

### 3. /analyze -- วิเคราะห์โพสต์ (ฟีเจอร์หลัก)

เขียนโพสต์เสร็จแล้ว วางเนื้อหาเพื่อวิเคราะห์ 4 มิติ:

```
/analyze

[วางเนื้อหาโพสต์ของคุณ]
```

4 มิติการวิเคราะห์:

- **เทียบสไตล์**: เปรียบเทียบกับสไตล์ย้อนหลังของคุณเอง ชี้จุดที่เบี่ยงเบนและผลงานย้อนหลัง
- **วิเคราะห์จิตวิทยา**: กลไก Hook, เส้นอารมณ์, แรงจูงใจในการแชร์, สัญญาณความน่าเชื่อถือ, อคติทางความคิด, ศักยภาพในการกระตุ้นคอมเมนต์
- **ตรวจอัลกอริทึม**: สแกนเส้นแดง (ตรงกันเมื่อไหร่เตือนทันที) + ประเมินสัญญาณบวก
- **ตรวจจับ AI**: สแกนร่องรอย AI ระดับประโยค, ระดับโครงสร้าง, ระดับเนื้อหา

### 4. /topics -- แนะนำหัวข้อ

ไม่รู้จะเขียนอะไรต่อ ใช้ตัวนี้ ขุดข้อมูลจากคอมเมนต์และข้อมูลย้อนหลังเพื่อแนะนำหัวข้อ

```
/topics
```

แนะนำ 3-5 หัวข้อ แต่ละหัวข้อมีแหล่งที่มาของคำแนะนำ, เหตุผลจากข้อมูล, ผลงานของโพสต์ย้อนหลังที่คล้ายกัน, ช่วงผลลัพธ์ที่คาดการณ์

### 5. /draft -- ช่วยร่างโพสต์

เลือกหัวข้อจาก topic bank และสร้างร่างโพสต์ตาม Brand Voice ของคุณ เป็นฟีเจอร์ AI เขียนโพสต์ ที่ตรงที่สุดของ AK-Threads-Booster แต่ร่างเป็นแค่จุดเริ่มต้น

```
/draft [หัวข้อ]
```

ระบุหัวข้อเองได้ หรือปล่อยให้ระบบแนะนำจาก topic bank คุณภาพร่างขึ้นอยู่กับความสมบูรณ์ของข้อมูล Brand Voice ถ้ารัน `/voice` มาก่อนจะเห็นความแตกต่างชัดเจน

ร่างเป็นแค่จุดเริ่มต้น คุณต้องแก้ไขปรับปรุงเอง หลังแก้ไขแนะนำให้รัน `/analyze`

### 6. /predict -- ทำนายผลลัพธ์

เขียนโพสต์เสร็จแล้ว ทำนายผลลัพธ์ 24 ชั่วโมงหลังโพสต์

```
/predict

[วางเนื้อหาโพสต์ของคุณ]
```

ให้ผลลัพธ์ 3 ช่วง: conservative / baseline / optimistic (views / likes / replies / reposts / shares) พร้อมเหตุผลและปัจจัยที่ไม่แน่นอน

### 7. /review -- รีวิวหลังโพสต์

หลังโพสต์แล้ว ใช้เพื่อเก็บข้อมูลผลลัพธ์จริง เทียบกับที่ทำนายไว้ และอัปเดตข้อมูลระบบ

```
/review
```

สิ่งที่ทำ:
- เก็บข้อมูลผลลัพธ์จริง
- เทียบกับที่ทำนาย วิเคราะห์สาเหตุที่แตกต่าง
- อัปเดต tracker และคู่มือสไตล์
- แนะนำเวลาโพสต์ที่ดีที่สุด

## Knowledge Base

AK-Threads-Booster มี knowledge base 3 ชุดในตัว ใช้เป็นฐานอ้างอิงในการวิเคราะห์

### จิตวิทยาโซเชียลมีเดีย (psychology.md)

แหล่งข้อมูล: รวบรวมจากงานวิจัย ครอบคลุมกลไกกระตุ้นจิตวิทยาของ Hook, จิตวิทยาการกระตุ้นคอมเมนต์, แรงจูงใจในการแชร์และกลไกการแพร่กระจาย (STEPPS framework), การสร้างความน่าเชื่อถือ (Pratfall Effect, Parasocial Relationship), การใช้อคติทางความคิด (Anchoring, Loss Aversion, Social Proof, IKEA Effect), เส้นอารมณ์และระดับการกระตุ้น

ใช้เพื่อ: เป็นฐานทฤษฎีสำหรับมิติวิเคราะห์จิตวิทยาใน `/analyze` จิตวิทยาเป็นเลนส์วิเคราะห์ ไม่ใช่กฎการเขียน

### อัลกอริทึม Meta (algorithm.md)

แหล่งข้อมูล: เอกสารสิทธิบัตร Meta, Facebook Papers, ประกาศนโยบายอย่างเป็นทางการ, การสังเกตจาก KOL (เสริมเท่านั้น) ครอบคลุมรายการเส้นแดง (12 พฤติกรรมที่ถูกลดการเข้าถึง), สัญญาณการจัดอันดับ (การแชร์ทาง DM, คอมเมนต์เชิงลึก, เวลาที่อยู่ในโพสต์ ฯลฯ), กลยุทธ์หลังโพสต์, กลยุทธ์ระดับบัญชี

ใช้เพื่อ: เป็นฐานกฎสำหรับการตรวจอัลกอริทึมใน `/analyze` เส้นแดงตรงกันเมื่อไหร่เตือนทันที สัญญาณอื่นให้ในโทนแนะนำ

### ตรวจจับ AI (ai-detection.md)

ครอบคลุมร่องรอย AI ระดับประโยค (10 ประเภท), ร่องรอย AI ระดับโครงสร้าง (5 ประเภท), ร่องรอย AI ระดับเนื้อหา (5 ประเภท), วิธีลดความเป็น AI (7 วิธี), เงื่อนไขการสแกน, คำจำกัดความความรุนแรง

ใช้เพื่อ: เป็นฐานตรวจจับสำหรับการสแกน AI ใน `/analyze` ชี้ร่องรอย AI ให้คุณแก้เอง ไม่แก้ไขอัตโนมัติ

## คำถามที่พบบ่อย

**Q: AK-Threads-Booster เขียนโพสต์แทนเราเลยไหม?**
โมดูล `/draft` สร้างร่างโพสต์ได้ แต่ร่างเป็นแค่จุดเริ่มต้น คุณต้องแก้ไขปรับปรุงเอง คุณภาพร่างขึ้นอยู่กับความสมบูรณ์ของข้อมูล Brand Voice โมดูลอื่นวิเคราะห์และแนะนำเท่านั้น ไม่เขียนแทน

**Q: ข้อมูลน้อย วิเคราะห์แม่นไหม?**
ตรงๆ คือยังไม่ค่อยแม่น ระบบจะบอกตรงๆ แบบนั้น แต่ยิ่งสะสมข้อมูลมากขึ้น ก็ยิ่งแม่นขึ้น

**Q: ต้องทำตามคำแนะนำเสมอไหม?**
ไม่จำเป็น ทุกข้อแนะนำเป็นแค่ข้อมูลประกอบการตัดสินใจ คุณมีสิทธิ์ตัดสินใจสุดท้ายเสมอ สิ่งเดียวที่เตือนตรงๆ คือเส้นแดงอัลกอริทึม (รูปแบบที่ทำให้ถูกลดการเข้าถึง)

**Q: รองรับแพลตฟอร์มอื่นนอกจาก Threads ไหม?**
ตอนนี้ออกแบบมาสำหรับ Threads เป็นหลัก หลักจิตวิทยาใน knowledge base ใช้ได้ทั่วไป แต่ knowledge base อัลกอริทึมเน้น Meta โดยเฉพาะ

**Q: ต่างจากเครื่องมือ AI เขียนโพสต์ทั่วไปยังไง?**
เครื่องมือทั่วไปสร้างคอนเทนต์จากโมเดลกลาง การวิเคราะห์และคำแนะนำของ AK-Threads-Booster ทั้งหมดมาจากข้อมูลย้อนหลังของคุณเอง ผลลัพธ์ของแต่ละคนจึงไม่เหมือนกัน เป็นที่ปรึกษา ไม่ใช่คนเขียนแทน

**Q: ช่วยเรื่องเวลาโพสต์ที่ดีที่สุดได้ไหม?**
ได้ โมดูล `/review` วิเคราะห์ข้อมูลผลลัพธ์จริงของคุณและแนะนำเวลาโพสต์ที่เหมาะสม ข้อมูลมาจากบัญชีของคุณเอง ไม่ใช่สถิติทั่วไป

**Q: ช่วยเรื่อง เพิ่มผู้ติดตาม Threads ได้โดยตรงไหม?**
AK-Threads-Booster ไม่ใช่เครื่องมือเพิ่มผู้ติดตามอัตโนมัติ สิ่งที่ทำคือวิเคราะห์และแนะนำเพื่อให้แต่ละโพสต์ทำผลงานได้ดีขึ้น การเพิ่มผู้ติดตามเป็นผลลัพธ์จากการโพสต์คอนเทนต์ดีๆ อย่างสม่ำเสมอ

**Q: ใช้แล้วการันตีโพสต์ปังไหม?**
ไม่การันตี อัลกอริทึมของ Threads เป็นระบบที่ซับซ้อนมาก ไม่มีเครื่องมือไหนการันตีโพสต์ปังได้ สิ่งที่ AK-Threads-Booster ทำคือช่วยให้คุณตัดสินใจได้ดีขึ้นจากข้อมูลย้อนหลังของคุณเอง หลีกเลี่ยงเส้นแดงของอัลกอริทึมที่รู้แล้ว และเพิ่มโอกาสที่แต่ละโพสต์จะทำผลงานได้ดีจากมุมจิตวิทยาและการวิเคราะห์ข้อมูล เป็น Threads Skill ที่ครบถ้วนที่สุดในตอนนี้ แต่ปัจจัยที่กำหนดว่าโพสต์จะปังไหม ไม่ว่าจะเป็นจังหวะเวลา ความเกี่ยวข้องของหัวข้อ สถานะของกลุ่มเป้าหมาย ตรรกะการกระจายของอัลกอริทึมในขณะนั้น ไม่มีเครื่องมือไหนควบคุมได้ทั้งหมด ใช้เป็นที่ปรึกษาข้อมูล ไม่ใช่เครื่องการันตีโพสต์ปัง
